ExtendedModalWindow

Class in TNRD.Editor.Core

Description

Base class for modal windows that can be added to ExtendedWindows

Fields

Editor The editor this modal window is shown in
shouldAlignToCenter Should the modal window be aligned to the center of the editor
showCancelButton Should the modal window show the Cancel/Close button
showOKButton Should the modal window show the OK/Accept button
textCancelButton The text to show on the Cancel/Close button
textOKButton The text to show on the OK/Accept button
Title The title of the modal window
WindowRect The rectangle used to draw the modal window
WindowStyle The GUIStyle for the modal window

Properties

Input The input manager
IsDone Is the modal window ready to close
IsDraggable Can the modal window be dragged around
Result The result of a closed modal window

Constructors

ExtendedModalWindow Creates an instance of ExtendedModalWindow

Methods

AlignToCenter Aligns the modal window to the center of the editor
Cancel Closes the modal window with the Cancel result
OK Closes the modal window with the OK result

OnDestroy Called when the modal window gets closed
OnDragExited Invoked when a DragExited event occurs
OnDragPerform Invoked when a DragPerform event occurs
OnDragUpdate Invoked when a DragUpdate event occurs
OnGUI Implement your own GUI logic here
OnInitialize Called the first time OnGUI is called on this modal window
