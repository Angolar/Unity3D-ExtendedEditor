Scripting API


ExtendedInput

Class in TNRD.Editor.Core

Description

The input manager for every editor

Fields

MousePosition The current mouse position in screen coordinates

Properties

Button The mouse button that invoked a double click
IsDoubleClick Did a double click occur. Check which button with \"Button\"
MouseDelta The current mouse delta in the editor/window (including scrollwheel delta)
MouseWorldPosition The current mouse position in world coordinates
ScrollDelta The current scrollwheel delta
Type The current event that is being processed

Methods

ButtonDown Is the given mouse button down
ButtonPressed Did the given mouse button get pressed this frame
ButtonReleased Did the given mouse button get released this frame
ButtonUp Is the given mouse button up
KeyDown Is one of the given keys down
KeyPressed Did one of the given keys get pressed this frame
KeyReleased Did one of the given keys get released this frame
KeysDown Are all of the given keys down
KeysPressed Did all of the given keys get pressed this frame
KeysReleased Did all of the given keys get released this frame
KeysUp Are all of the given keys up
KeyUp Is one of the given keys up
OnGUI Handles the input events
Reset Resets all the key and mousebutton states
Update Called 100 times per second
