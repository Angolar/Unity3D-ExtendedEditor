Scripting API


ExtendedEditor

Class in TNRD.Editor.Core

Description

The base for every editor made with the Extended Editor framework

Fields

DeltaTime The time in seconds it took to complete the last frame
RepaintOnUpdate If true the editor will try to repaint every update
SharedAssets An asset manager that can be accessed from every window
SharedObjects The active shared objects in this editor
Windows The active windows in this editor

Properties

CurrentEvent The current event being processed by every input manager
Input The input handler for the editor
