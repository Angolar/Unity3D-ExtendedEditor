Scripting API


ExtendedControl

Class in TNRD.Editor.Core

Description

Base class for controls that can be added to ExtendedWindows

Fields

AllowDrop Gets or sets a value indicating whether the control can accept data that the user drags onto it
Anchor Gets or sets the edges of the window to which a control is bound and determines how a control is resized with its parent.
IsInitialized Is the control initialized
MaxSize The upper limit for the size of this control. To be used in combination with Anchor
MinSize The lower limit for the size of this control. To be used in combination with Anchor
Position The position of the control in the window
Size The size of the control
Window The window this control is added to

Properties

ExecutionOrder Defines where along the line of all the controls this control gets its Update and GUI calls
Input The input handler belonging to this control's window
Rectangle The rectangle used for drawing in OnGUI

Constructors

ExtendedControl Creates a new instance of ExtendedControl

Methods

GetControlID Gets a unique ID for a control
OnContextClick Invoked when a ContextClick event occurs
OnDeserialized Called when a window gets deserialized
OnDestroy Called when the Window this control is located in gets closed
OnDragExited Invoked when a DragExited event occurs
OnDragPerform Invoked when a DragPerform event occurs
OnDragUpdate Invoked when a DragUpdate event occurs
OnGUI Implement your own GUI logic here
OnInitialize Called when the control is added to a window
OnInitializeGUI Called the first time OnGUI is called on this control
OnSceneGUI Implement your own SceneGUI logic here
